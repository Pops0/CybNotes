[[Web Security]]

**OWASP Juice Shop** is one of the **most popular web-hacking practice apps**.

What you can practice:

- SQL injection
    
- XSS
    
- authentication bypass
    
- broken access control
    

Run it:

```
docker run -d -p 3000:3000 bkimminich/juice-shop
```

Open:

http://localhost:3000

This is widely used in:

- pentesting courses
    
- CTFs
    
- bug bounty training
